# Folder Structure

Shopify shell follows the following directory pattern. 

├── assets
│   ├── favicon
│   ├── fonts
│   ├── images
│   └── svg
├── config
├── design-tokens
├── js / bundles
│   ├── components
│   ├── layout
│   ├── sections
│   └── templates
│       └── customers
├── liquid
│   ├── layout
│   ├── sections
│   ├── snippets
│   └── templates
│       └── customers
├── locales
└── styles
    ├── components
    │   ├── global
    │   └── shared
    ├── layout
    ├── sections
    └── templates
        └── customers
